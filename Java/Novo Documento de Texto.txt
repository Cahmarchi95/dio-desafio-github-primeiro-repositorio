Fases execução Java
arquivo.extensão java > JDK compila o arquivo > bytecode arquivo extensão.class > JVM lê o arquivo extensão.class e bibliotecas que estão no JRE

JDK (Java Development kit): compila o código fonte (.java) em bytecode (.class)

JVM (Java Virtual Machine): é a máquina virtual do Java que executa o bytecode (.class) 

JRE (Java runtime environment): ambiente de execução do Java que fornece as bibliotecas padrões do Java para o JDK compilar o código e para o JVM executar o programa
